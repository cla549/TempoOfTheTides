# CMPT276S26_Group13

Tempo of the Tides is a tile-based rhythm game where a robber attempts to escape
a pirate ship with as much loot as possible. The catch: in order to keep his
footing, he needs to walk in time with the waves, which conveniently matches up
perfectly with the pirates' strange taste in electronic sea shanties.

How much loot can you plunder in time with the tempo of the tides?
